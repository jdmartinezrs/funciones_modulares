# Reto 1 - Sistema de Cálculo Académico

## Descripción
Este reto consiste en crear un programa modularizado que permita registrar el nombre y tres notas de un estudiante para determinar su estado académico.

## Objetivos de Aprendizaje
- Implementar modularización mediante funciones
- Aplicar retorno de valores entre funciones
- Utilizar estructuras condicionales
- Manejo de parámetros y argumentos
- Interacción básica con el usuario

## Especificaciones
Crea un programa que permita:
- Registrar el nombre de un estudiante
- Ingresar tres notas académicas
- Calcular el promedio de las notas
- Determinar si el estudiante aprueba o reprueba
- Mostrar un mensaje final con el resultado

## Requisitos Técnicos
- Cada funcionalidad debe estar en una función diferente
- Las funciones deben comunicarse mediante retorno de valores
- Aplicar estructuras condicionales para la lógica de aprobación
- El código debe ser modular y reutilizable
