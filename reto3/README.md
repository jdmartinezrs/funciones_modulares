# Reto 3 - Sistema de Facturación

## Descripción
Este reto consiste en crear un programa que reciba una lista de productos (nombre, precio, cantidad) y calcule los diferentes componentes de una factura: subtotal por producto, total sin IVA, IVA total (19%) y valor total a pagar.

## Objetivos de Aprendizaje
- Implementar modularización mediante funciones
- Aplicar retorno de valores entre funciones
- Manejo de arrays/listas de productos
- Realizar cálculos matemáticos para facturación
- **Desafío adicional**: Implementar funciones flecha
- Estructuras de datos para representar productos

## Especificaciones
Crea un programa que permita:
- Recibir una lista de productos con nombre, precio y cantidad
- Calcular el subtotal por producto
- Calcular el total sin IVA
- Calcular el IVA total (19%)
- Calcular el valor total a pagar
- **Desafío**: Implementar la función final como función flecha

## Requisitos Técnicos
- Cada cálculo debe estar en una función diferente
- Las funciones deben comunicarse mediante retorno de valores
- Manejo de estructuras de datos para productos
- Cálculos precisos de porcentajes
- La función final debe ser una función flecha
- El código debe ser modular y reutilizable
