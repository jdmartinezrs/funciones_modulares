# Reto 2 - Módulo de Operaciones Bancarias

## Descripción
Este reto consiste en diseñar un sistema que permita simular operaciones bancarias básicas: depositar dinero, retirar dinero y consultar saldo, manteniendo el saldo actualizado entre operaciones.

## Objetivos de Aprendizaje
- Implementar modularización mediante funciones
- Mantener estado entre operaciones (saldo)
- Aplicar estructuras de ciclo para operaciones repetitivas
- Utilizar condicionales para validaciones
- Manejo de parámetros y argumentos
- Interacción continua con el usuario

## Especificaciones
Crea un sistema que permita:
- Depositar dinero en una cuenta
- Retirar dinero de una cuenta (validando fondos suficientes)
- Consultar el saldo actual
- Realizar múltiples operaciones en secuencia
- Mantener el saldo actualizado entre operaciones
- Salir del sistema cuando el usuario lo decida

## Requisitos Técnicos
- Cada operación debe estar en una función diferente
- El saldo debe mantenerse actualizado entre operaciones
- Debe haber un ciclo que permita múltiples operaciones
- Validación de fondos suficientes para retiros
- El código debe ser modular y reutilizable
