# Proyecto de Funciones Modulares

Este proyecto contiene una serie de retos de programación enfocados en el desarrollo de habilidades en modularización, funciones y buenas prácticas de programación.

## Estructura del Proyecto

El proyecto está organizado en carpetas, una por cada reto, donde cada desarrollador encuentra las especificaciones detalladas para implementar cada algoritmo.

### Retos Incluidos

- **reto1/** - Sistema de Cálculo Académico
- **reto2/** - Módulo de Operaciones Bancarias  
- **reto3/** - Sistema de Facturación
- **reto4/** - Análisis de Números
- **reto5/** - Reto Creativo

## Contenido de Cada Reto

Cada carpeta de reto contiene:

1. **README.md** - Descripción general del reto, objetivos de aprendizaje y criterios de evaluación
2. **[nombre_algoritmo].md** - Especificaciones detalladas del algoritmo, incluyendo funciones requeridas, parámetros, retorno y criterios de aceptación

## Cómo Usar Este Proyecto

1. **Selecciona un reto** - Revisa los 5 retos disponibles y elige uno para comenzar
2. **Lee las especificaciones** - Entra en la carpeta del reto y lee primero el README.md para entender el objetivo general
3. **Revisa el detalle** - Lee el archivo de especificaciones del algoritmo para conocer los requisitos técnicos detallados
4. **Implementa la solución** - Desarrolla el algoritmo siguiendo las especificaciones
5. **Verifica tu solución** - Asegúrate de cumplir con todos los criterios de aceptación

## Objetivos de Aprendizaje Generales

- Dominio de funciones tradicionales y funciones flecha
- Comprensión profunda de parámetros y argumentos
- Aplicación correcta de retorno de valores
- Modularización del código
- Buenas prácticas de programación
- Desarrollo de algoritmos estructurados

## Convenciones de Nombres

Los archivos de especificaciones tienen nombres afines a cada algoritmo para facilitar su identificación:

- `sistema_calculo_academico.md` - Reto 1
- `modulo_operaciones_bancarias.md` - Reto 2
- `sistema_facturacion.md` - Reto 3
- `analisis_numeros.md` - Reto 4
- `reto_creativo.md` - Reto 5

## Colaboración

Este proyecto está diseñado para el trabajo colaborativo entre desarrolladores. Cada reto puede ser abordado independientemente, permitiendo que diferentes desarrolladores trabajen en paralelo.

## Notas Importantes

- Lee completamente las especificaciones antes de comenzar a implementar
- Sigue los criterios de aceptación detallados en cada especificación
- Aplica las buenas prácticas de programación mencionadas
- Documenta tu código cuando sea necesario
- Verifica que tu solución cumpla con todos los requisitos antes de considerarla completa
